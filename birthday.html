🛠 How to personalise this page

  1. Open this file in any text editor (Notepad, VS Code, etc.)
  2. Search for Your Special Day and replace with the real date
  3. To add your friend's name, find the <h1 class="hero-name"> tag and type the name under "Happy Birthday"
  4. Click any photo placeholder on the page to upload a photo from your device
  5. Edit any of the captions or wish texts directly in the HTML
  6. To host it: push to GitHub → enable GitHub Pages → share the link 🎉
